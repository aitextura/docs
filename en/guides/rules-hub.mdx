**Rules Hub** is a centralized storage of instruction blocks that you can use when configuring the Persona of any AI employee. Instead of writing instructions from scratch every time, you pick a ready-made block from the hub, insert it into a [Persona](/en/guides/persona), and adapt it to your needs.

The Rules Hub is located in the left menu of the platform. Clicking it reveals a submenu with two sections: **Templates** and **My Rules**.

## Rules Hub Structure

The hub consists of two sections, each accessible both from the left menu and via the **Apply Template** button inside the Persona editor.

| Section | Contents | Created by |
|---|---|---|
| **Templates** | Pre-built blocks for popular roles and scenarios | AI Textura team |
| **My Rules** | Blocks you saved from an employee's Persona | You |

Both sections work the same way: find the block you need, click on it, and it gets added to the selected employee's Persona. You can then edit the block to fit your specific task.

## Templates — Pre-built Blocks from AI Textura

**Templates** is a library of professionally written blocks covering typical roles and scenarios. The AI Textura team created them based on real use cases so you can launch an employee quickly without lengthy setup.

### Block Types

Each template belongs to one of the block types used in a Persona:

| Block type | What it describes | Example templates |
|---|---|---|
| **Role** | Who your employee is, what role they play | Hotel administrator, Realtor, Personal assistant |
| **Tone** | How the employee speaks: style, tone, form of address | Friendly tone, Formal tone |
| **Instructions** | Specific behavioral rules and reactions | Greeting, Prohibited actions, Escalation |
| **Script** | Step-by-step scenarios for typical tasks | Booking (hotel), Real estate sales |

### Finding Templates

Two tools are available in the **Templates** section:

- **Search by name** — enter a keyword, e.g. "hotel" or "booking"
- **"All types" filter** — select a specific block type (Role, Tone, Script, etc.) to narrow down the list

### How to Apply a Template

<Steps>
  <Step title="Option 1 — via Rules Hub (left menu)">
    1. Open **Rules Hub** → **Templates** in the left menu
    2. Find the desired template using search or the filter
    3. Click on the template
    4. Select the employee you want to apply it to
    5. The block will be added to the employee's Persona
  </Step>
  <Step title="Option 2 — from the Persona editor (quick access)">
    1. Open the desired employee → **Persona** tab
    2. Click the **Apply Template** button
    3. In the dialog, select the **Templates** tab
    4. Find the template and click on it — the blocks will be added to the Persona
    5. Edit the text to match your specifics
    6. Save your changes
  </Step>
</Steps>

<Tip>
  Templates are a starting point, not a finished solution. Always adapt the text to your business: replace names, add specific details, remove what is unnecessary. The more precisely an instruction describes your situation, the more predictably the employee will perform.
</Tip>

## My Rules — Your Personal Library

**My Rules** is the section that stores blocks you saved from any employee's Persona. It is your personal library of reusable instructions.

Typical use cases:

- Identical rules for multiple employees (e.g., "do not make up information" or "do not ask more than two questions in a single message")
- A universal communication tone applied to all company employees
- Proven scenarios that you refined with one employee and want to replicate

### How to Save a Block to "My Rules"

<Steps>
  <Step title="Open the employee's Persona">
    Navigate to the desired employee and open the **Persona** tab.
  </Step>
  <Step title="Save the block">
    Find the block you want to save, click the **"..."** icon (block menu) → **Save**.
  </Step>
  <Step title="Block is available in the Hub">
    The block will appear in the **My Rules** section — both in the left menu and in the **Apply Template** dialog inside the Persona editor.
  </Step>
</Steps>

### How to Apply a Saved Block to Another Employee

<Steps>
  <Step title="Open the target employee's Persona">
    Navigate to the desired employee → **Persona** tab.
  </Step>
  <Step title="Open templates">
    Click **Apply Template** → switch to the **My Rules** tab.
  </Step>
  <Step title="Apply and adapt">
    Click on the desired block — it will be added to the Persona. Edit the text if needed and save your changes.
  </Step>
</Steps>

<Note>
  Rules in the "My Rules" section are grouped by type: Role, Tone, Script, custom. Use the "All types" filter and search by name to quickly find blocks, especially when you have many saved.
</Note>

### Creating a Rule Directly in the Hub

In addition to saving blocks from a Persona, you can create a new rule directly in the Hub — without tying it to a specific employee:

1. Open **Rules Hub** → **My Rules** in the left menu
2. Click the **Add Rule** button
3. Set a name and block type
4. Write the instruction text
5. Save

The created rule will immediately appear in your library and be available for use with any employee.

## "/" Command — Quick Import from the Editor

In the Persona editor, blocks from the Hub can be inserted via the **/** command at the end of the editor:

| Command | What it does |
|---|---|
| **/** → **Import from hub** → **My Rules** | Inserts your saved block |
| **/** → **Import from hub** → **Public** | Inserts a template from the AI Textura team |

## Entry Points to the Rules Hub

| From | Where you land | What you see |
|---|---|---|
| Left menu → **Rules Hub** → **Templates** | Hub page, "Templates" tab | All public templates from the AI Textura team |
| Left menu → **Rules Hub** → **My Rules** | Hub page, "My Rules" tab | Your saved blocks |
| Persona editor → **Apply Template** | Pop-up dialog with two tabs | Same data: Templates and My Rules |
| Persona editor → **/** → **Import from hub** | Source selection: My Rules / Public | Quick block insertion into the current Persona |

<Warning>
  Blocks from the Rules Hub are inserted into a Persona **as a copy**. If you later modify the block in the Hub, it will not affect copies already inserted into employee Personas. Each copy must be updated separately.
</Warning>

## FAQ

<Accordion title="What is the difference between 'Templates' and 'My Rules'?">
  Templates are pre-built blocks from the AI Textura team, available to all users. My Rules are your own blocks that you saved or created. Templates cannot be edited in the Hub — only after being applied to an employee. My Rules are fully under your control.
</Accordion>

<Accordion title="If I change a block in 'My Rules', will it update for all employees?">
  No. When you apply a block from the Hub to an employee, a copy is inserted into the Persona. Changes to the original do not affect already inserted copies, and vice versa. If you need to update a block across multiple employees, you must do it manually for each one.
</Accordion>

<Accordion title="Can I use a single template for multiple employees?">
  Yes. Apply the template to one employee, then to another — each receives their own copy that can be edited independently.
</Accordion>

<Accordion title="Is there a limit on the number of saved rules?">
  No. You can save as many blocks as you need. We recommend giving blocks clear names and using filters to stay organized.
</Accordion>

<Accordion title="Can I delete a block from 'My Rules'?">
  Yes. Deleting a block from the Hub will not affect employees that already have it applied — they retain their own copy.
</Accordion>

<Accordion title="Does the order of applied templates affect employee behavior?">
  Yes. Blocks positioned higher in the Persona have greater priority. After applying a template, you can drag blocks into the desired order directly in the editor.
</Accordion>
